Quero criar um site completo para um campeonato de League of Legends. O site deve ser moderno, responsivo e otimizado para desktop, tablet e celular. Preciso que o design seja inspirado em temas de eSports, com cores escuras, neon (azul e roxo) e animações sutis. O site deve ter as seguintes funcionalidades: sistema de login e cadastro de usuários com autenticação via email usando Supabase, diferenciando dois tipos de usuário: capitães e jogadores. Os capitães poderão acessar uma área especial para o sorteio de times, onde cada capitão escolhe uma lane (Top, Jungle, Mid, ADC e Suporte) e o sistema sorteia de forma totalmente aleatória um jogador daquela lane, a partir de um pool de 20 jogadores pré-cadastrados (4 por lane). A página de sorteio deve ter uma animação elaborada com cartas de jogadores, no estilo cinematográfico inspirado em eFootball, onde uma carta brilha, gira e revela o jogador escolhido, com efeitos visuais de estádio, neon e explosões de luz. Esse sorteio precisa ser exibido de forma rápida, mas impactante.

O site deve conter também as seguintes páginas: página inicial com destaque para o campeonato, notícias e atualizações; página de regras do torneio; página “Onde Assistir” com links para transmissão ao vivo (Twitch e YouTube); página de histórico das partidas (com a possibilidade futura de integração com a API da Riot Games para puxar dados automaticamente, como kills, ouro, tempo de jogo etc); página de jogadores com perfis individuais (nome, lane, histórico e foto); página de times mostrando as formações após o sorteio; e uma página sobre o campeonato, explicando o projeto e os organizadores.

Preciso que o site tenha um painel administrativo simples para que os organizadores possam atualizar notícias, resultados, informações dos times e jogadores. O design deve ser minimalista mas com toques de animação moderna (hover em botões, transições suaves entre páginas e alguns efeitos visuais em neon). Também quero que o site esteja preparado para receber SEO básico e compartilhamento em redes sociais (imagem de destaque, título e descrição automática ao compartilhar).

Garanta que todo o fluxo seja seguro, com autenticação via Supabase, e que os capitães só possam acessar a página de sorteio depois de logados e autorizados. O sistema deve estar pronto para expansão, permitindo no futuro adicionar outras funcionalidades, como estatísticas detalhadas puxadas da API da Riot.
